  <h1>✅ SGT - Sistema de Gestão de Tarefas</h1>

  <p>O <strong>SGT</strong> (Sistema de Gestão de Tarefas) é uma aplicação web desenvolvida em <code>HTML</code>, <code>CSS</code> e <code>JavaScript</code> puro que permite ao usuário organizar, visualizar e acompanhar suas tarefas de forma prática e intuitiva, sem necessidade de frameworks externos ou banco de dados.</p>

  <h2>📋 Funcionalidades</h2>
  <ul>
    <li>Cadastro de tarefas com:
      <ul>
        <li>Nome da tarefa</li>
        <li>Prioridade (Baixa, Média, Alta, Urgente)</li>
        <li>Data e hora de execução</li>
      </ul>
    </li>
    <li>Edição, conclusão e remoção de tarefas</li>
    <li>Visualização do histórico das últimas tarefas modificadas</li>
    <li>Notificações visuais para ações realizadas</li>
    <li>Interface responsiva para uso em dispositivos móveis</li>
  </ul>

  <h2>🛠️ Tecnologias Utilizadas</h2>
  <ul>
    <li><strong>HTML5</strong> para estruturação do conteúdo</li>
    <li><strong>CSS3</strong> para estilização e responsividade</li>
    <li><strong>JavaScript (ES6+)</strong> para lógica da aplicação e manipulação do DOM</li>
  </ul>

  <h2>⚙️ Como Usar</h2>
  <ol>
    <li>Abra o arquivo <code>index.html</code> no seu navegador</li>
    <li>Adicione uma nova tarefa preenchendo o formulário</li>
    <li>Gerencie tarefas (concluir, editar, remover) pelos botões correspondentes</li>
    <li>Visualize o histórico das últimas tarefas na seção dedicada</li>
  </ol>

  <h2>📌 Observações Técnicas</h2>
  <p>Este projeto não utiliza armazenamento local (localStorage) nem backend. Portanto, as tarefas não persistem após o fechamento ou atualização da página. O foco é demonstrar manipulação dinâmica do DOM, gerenciamento de estados no front-end e usabilidade básica sem dependências externas.</p>
